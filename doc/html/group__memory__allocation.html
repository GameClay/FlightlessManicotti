KL_API void* kl_heap_aligned_alloc ( size_t  size,
size_t  align_size 
)

Aligned malloc function.

Note:
The amount of allocated memory may exceed the size specified.
Attention:
This function will assert if the align_size == 16. If you want alignment on a 16-byte boundary use kl_heap_malloc, which returns 16-byte aligned memory at all times.
Parameters:
sizeThe size of the allocation.
align_sizeThe boundary on which the allocated memory should be aligned.
Returns:
Pointer the block of allocated memory, aligned on the specified boundary, or NULL if the allocation failed.

KL_API void* kl_micro_alloc ( size_t  size)

Micro malloc function.

Allocates small amounts of memory.

Note:
This is not assured to be a specalized micro-allocator. If KL_USE_MICROALLOCATOR is defined, it will John Ratcliff's microallocator.
Attention:
If the allocation size is greater than 256 bytes, this function will assert, and return NULL.
Parameters:
sizeThe size of the allocation.
Returns:
Pointer the block of allocated memory or NULL if the allocation failed.
